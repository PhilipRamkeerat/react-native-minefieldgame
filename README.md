# react-native-minefieldgame

A game created in React Native, about minefield game.


## To execute game
<pre>
<code>
npx react-native start
npx react-native run-android
</code>
</pre>

- For ios:
<pre>
<code>
npx react-native run-ios
</code>
</pre>

## Requirements
-Node stable version
-react native cli or use npx
